Lo aprendido en html5

En clases, aprendimos la definición de HTML, que significa HyperText Markup Language (Lenguaje de
de Hipertexto). Este lenguaje se utiliza para crear la estructura básica de todas las páginas web.

HTML funciona mediante etiquetas que se escriben entre corchetes angulares (< >). Estas etiquetas
indican al navegador web cómo interpretar y mostrar el contenido de la página.

En clases, no solo aprendimos la definición de HTML y sus usos, sino también cómo usar Visual Studio
Code para crear la estructura básica de una página web.

Visual Studio Code es un editor de código gratuito y de código abierto que ofrece muchas características
útiles para el desarrollo web, como resaltado de sintaxis, autocompletado y depuración.
